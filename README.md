# Data Structure & Algorithm Study Log

> **자료구조와 알고리즘, 원리와 성장의 기록**

이 저장소는 제가 직접 다양한 자료구조와 알고리즘을 공부하고, 구현하면서 느낀 점과 배운 내용을 정리한 공간입니다.  
단순히 코드를 작성하는 데 그치지 않고, ‘왜 이렇게 동작하는지’를 고민하며 작성했습니다.  
모든 코드는 C로 작성하였고, 테스트까지 거쳤습니다.

## 구현 목록

- **탐색**: 이진 탐색, 순차 탐색
- **그래프 탐색**: BFS, DFS
- **최단 경로**: 다익스트라, A* 알고리즘
- **최소 신장 트리**: Kruskal, Prim, 개선된 Prim
